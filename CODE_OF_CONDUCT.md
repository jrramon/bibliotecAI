# Code of Conduct

## Our commitment

We want BibliotecAI to be a place where anyone — regardless of level of
experience, background, or identity — can contribute, ask questions, and
learn. To that end, we adopt the [Contributor Covenant][cc], version 2.1,
as the code of conduct for this project.

## Summary

- Be respectful. Disagree with ideas, not people.
- Assume good faith. Ask before accusing.
- No harassment, insults, or discriminatory language — in issues, pull
  requests, commits, or any other project space.
- Maintainers may remove, edit, or reject contributions that do not align
  with this code of conduct.

## Reporting

If you experience or witness unacceptable behavior, please open a private
[GitHub Security Advisory][advisory] on this repo or email the maintainers
via the contact listed on the GitHub profile. Reports are handled
confidentially.

The full Contributor Covenant text is available at
<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>.
